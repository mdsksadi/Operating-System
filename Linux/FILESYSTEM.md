# FILESYSTEM
| File Name | Contains          |    
|-----------|-------------------|   
| **/bin**  | User Binaries     |   
| **/sbin** | System Binaries   |   
| **/etc**  | Config Files      |   
| **/dev**  | Device Files      |      
| **/proc** | Process Files     |
| **/var**  | Variable Files    |   
| **/tmp**  | Temparary Files   |   
| **/usr**  | User Program      |   
| **/home** | Home Directory    |   
| **/boot** | Boot Files        |   
| **/lib**  | System Libraries  |   
| **/opt**  | Optional App      |   
| **/mnt**  | Mount Directory   |   
| **/media**| Removable Media   |   
| **/srv**  | Service Data      |
